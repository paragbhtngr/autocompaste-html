Instructions

Dear participant , please read the instructions before the experiment.

INTRODUCTION

We are comparing a new technique for copying and pasting information, called AutoComPaste, to traditional Copy-Paste techniques (Using Ctrl-C and Ctrl-V). The traditional Copy-Paste technique uses the following protocol:

  • Select the text you wish to copy
  • Hold “Ctrl” (or “Cmd” on a Mac) and press “C” to copy the text
  • Navigate to the box the text is to be pasted into
  • Hold “Ctrl” (or “Cmd” on a Mac) and press “V” to paste the text

The AutoComPaste technique works on a different principle. It does not require you to select the text you wish to copy from the window. Instead, the following protocol is used:

  • In the box that the text is supposed to be pasted into, start typing the text you wish to paste
  • After the first 3 characters, suggested paste options will show up. Keep typing to narrow the search results or select one of the options from the menu using the “up” / “down” keys on the keyboard.
  • Press enter to select the text to paste. In order to append the next sentence to the pasted text, press the “right” key on the keyboard. In order to remove the last sentence from the pasted text, press the “left” key.

EXPERIMENT

You will be given a several pieces of text of varying lengths to copy and paste using both traditional methods and AutoComPaste. In the panel on your left, the technique to be used will be listed beside the Technique table. The text to be selected will be highlighted in red on the table in the left.

If the trial is using the TRADITIONAL technique, search for the text to be pasted in the open windows (the text will be highlighted in yellow), copy the text using the traditional technique and paste it in the text editor window. Once you are done, press the blue “Next Task” button to move onto the next trial.

The first six trials are meant to be for practice, to allow you to get used to the experiment, following which there will be a block of 54 trials, after which you will have a break of 1-2 mins, followed by 2 more blocks with a 1-2 min interval after each block. When you are ready to continue with the experiment, please press the blue “Next Block” button.

Thank you!
